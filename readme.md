Cantina Fatec

Projeto acadêmico desenvolvido para as disciplinas Estrutura de Dados e Linguagem de Programação 2 da Fatec Rio Claro.



Objetivo do Projeto

Este sistema tem como objetivo simular o funcionamento de uma cantina universitária gerenciada pela Atlética, permitindo o controle eficiente de:

- Estoque de produtos perecíveis
- Pagamentos realizados via PIX
- Consumo de alunos, professores e servidores

Além disso, o projeto reforça conceitos fundamentais de estruturas de dados, encapsulamento e organização de código orientado a objetos.
